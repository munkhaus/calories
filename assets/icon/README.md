# App Icon Design Instructions

## Description
Create a 1024x1024 PNG image named `app_icon.png` that represents the calorie tracking app's capabilities.

## Design Elements
The icon should include:
- Food/nutrition element (fork & knife, apple, or plate)
- Activity/fitness element (running figure or dumbbell)
- Tracking/analytics element (chart or graph)
- Camera element (small camera icon for photo feature)

## Color Scheme
Use the app's color palette:
- Primary: #6366F1 (purple-blue)
- Secondary: #EC4899 (pink)
- Warning: #F59E0B (orange/yellow)
- Success: #10B981 (green)

## Style Guidelines
- Modern, flat design
- Clear and recognizable at small sizes
- Avoid too much detail
- Use gradients between primary colors
- Rounded corners/modern aesthetic

## Suggested Layout
- Central plate/circle with food elements
- Small running figure in corner
- Small camera icon
- Background gradient from primary to secondary colors

## After Creating the Icon
1. Save as `app_icon.png` in this directory
2. Run: `flutter pub get`
3. Run: `flutter pub run flutter_launcher_icons:main`
4. Build the app to see the new icon

## Alternative: Use Online Icon Generator
You can use Canva, Figma, or other design tools to create the icon based on these specifications.
