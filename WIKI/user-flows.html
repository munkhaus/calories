📸 Foto Analyse Flow

Detaljeret flow for AI-assisteret mad registrering via billeder.

⚠️

LLM Cost Alert

Denne flow bruger GeminiService som koster $0.38-0.90 per billede. Se LLM Elimination Plan for gratis alternativer.

1

Image Capture

Service: CameraService

  • Camera: ImagePicker med quality optimization
  • Gallery: Existing image selection
  • Storage: Save til app documents/food_images/
  • Multi-image: Support op til 5 billeder per måltid
2

Pending Food Creation

Service: PendingFoodService

  • Create PendingFoodModel med image paths
  • Background AI analysis start
  • Add til pending foods list
  • User notification om pending analysis
Background: AI analyse kører asynkront
3

AI Analysis Process

Service: GeminiService

  • Single image: analyzeFoodImage()
  • Multiple images: analyzeMultipleFoodImages()
  • Output: FoodAnalysisResult med struktureret data
  • Error handling: Fallback til manual entry
AI Output: foodName, calories, protein, carbs, fat, confidence
4

User Review & Edit

Page: CategorizeFoodPage

  • Display AI suggestions
  • Allow manual edits til alle felter
  • Meal type auto-detection
  • Re-analyze button hvis nødvendigt
5

Final Confirmation

  • Convert til UserFoodLogModel
  • Log via FoodLoggingNotifier
  • Clean up pending food
  • Option: Save som favorite
Result: → Dashboard med logged meal

🏃‍♂️ Aktivitet Tracking Flow

Flow for registrering af motion og fysiske aktiviteter.

1

Activity Entry Points

Sources: Dashboard FAB | Activity Favorites | Quick Registration

  • Quick Activity: → QuickActivityRegistrationPage
  • Detailed Activity: → DetailedActivityRegistrationPage
  • Favorites: → ActivityFavoritesPage
2

Activity Registration

Models: UserActivityLogModel

  • Activity type: Running, cycling, gym, walking, etc.
  • Duration: Timer eller manual input
  • Intensity: Low, medium, high
  • Notes: Optional beskrivelse
3

Calorie Calculation

Service: ActivityNotifier

  • MET values for different activities
  • User weight fra profile
  • Duration and intensity modifiers
  • Auto-calculate burned calories
Formula: Calories = METs × weight(kg) × duration(h)
4

Activity Logging

  • Save til database
  • Update dashboard providers
  • Refresh calorie balance
  • Optional: Save som favorite
Updates: Today's Activities Widget + Calorie Overview
