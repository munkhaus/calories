💰 LLM Elimination Plan

Strategisk plan for at fjerne LLM afhængighed og reducere API omkostninger til $0/måned

💸 Nuværende Omkostninger: Appens LLM forbrug kan koste $790-1050/måned ved moderat brug (1000 søgninger + 500 billeder)

🎯 Nuværende LLM Forbrug

🔍 LLMFoodService

Placering: lib/features/food_database/infrastructure/llm_food_service.dart

  • Formål: Søgning i fødevaredatabase
  • Omkostning: ~$0.60-1.20 per søgning
  • API: Google Gemini 1.5 Flash
  • Rate Limit: 30 requests/minut

📸 GeminiService

Placering: lib/features/food_logging/infrastructure/gemini_service.dart

  • Formål: Analyse af fødevarebilleder
  • Omkostning: ~$0.38-0.90 per billede
  • Features: Enkelt/multi-billede analyse
  • Sprog: Dansk output

📋 Elimination Strategi

1

Uge 1: Erstat Food Search

Skift fra LLMFoodService til gratis Open Food Facts API

  • Implementer OpenFoodFactsService
  • Opdater provider registrering
  • Test søgefunktionalitet
2

Uge 2: Fjern Billede Analyse

Erstat GeminiService med forbedret manuel indtastning

  • Tilføj stemme-til-tekst input
  • Implementer smarte portionsvalg
  • Opret autocomplete søgning
3

Uge 3: Ryd Op

Fjern alle LLM afhængigheder

  • Slet LLM service filer
  • Fjern google_generative_ai dependency
  • Opdater imports og test grundigt
4

Uge 4: Forbedrede Features

Tilføj smart lokal funktionalitet

  • Kontekst-aware forslag
  • Lokal dansk fødevaredatabase
  • Offline backup funktionalitet

🔄 Erstatninger

📊 Open Food Facts API

Gratis erstatning for LLMFoodService med omfattende fødevaredatabase:

  • ✅ Gratis: Ingen API omkostninger
  • ✅ Omfattende: 2.7M+ produkter
  • ✅ Dansk support: Multi-sprog database
  • ✅ Høj rate limit: 100+ requests/minut
  • ✅ Struktureret data: Ingen LLM parsing nødvendig

Implementering:

class OpenFoodFactsService implements IOnlineFoodService {
  static const String _baseUrl = 'https://world.openfoodfacts.org/api/v2';
  
  @override
  Future<Result<List<OnlineFoodResult>, OnlineFoodError>> searchFoods(String query) async {
    final response = await http.get(
      Uri.parse('$_baseUrl/search?search_terms=$query&page_size=20&json=true'),
      headers: {'User-Agent': 'CaloriesApp-Flutter/1.0'},
    );
    // Struktureret parsing - ingen LLM nødvendig
  }
}
